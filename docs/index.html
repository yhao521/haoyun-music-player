📥 下载安装

选择适合您平台的版本

🍎

macOS

macOS 10.15+

  • ✅ Intel & Apple Silicon
  • ✅ DMG / ZIP 格式
  • ✅ Homebrew 安装
下载 DMG brew install --cask haoyun-music-player
🪟

Windows

Windows 10+

  • ✅ x86_64 & ARM64
  • ✅ NSIS 安装包
  • ✅ Chocolatey 安装
下载 EXE choco install haoyun-music-player
🐧

Linux

Ubuntu 20.04+

  • ✅ x86_64 & ARM64
  • ✅ DEB / RPM 包
  • ✅ AppImage
下载 DEB sudo dpkg -i haoyun-music-player.deb

从源码构建

# 克隆项目
git clone https://github.com/yhao521/haoyun-music-player.git
cd haoyun-music-player

# 安装依赖
go mod tidy
cd frontend && npm install && cd ..

# 运行应用
wails3 dev -config ./build/config.yml
