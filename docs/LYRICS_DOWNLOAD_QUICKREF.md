# 歌词下载功能 - 快速参考

## 🚀 快速开始

### 使用步骤
1. 确保已添加音乐库
2. 右键托盘图标 → **整理音乐** → **从网络下载歌词**
3. 等待后台下载完成
4. 查看通知结果

## 📋 核心 API

### 单首歌曲下载(多源降级)
```go
err := lyricManager.DownloadLyricWithFallback(trackPath, title, artist, album)
// 自动尝试: lrclib.net → 网易云 → QQ 音乐
```

### 批量下载
```go
success, fail, skip, errors := musicService.DownloadLyricsForLibrary()
```

## 🔧 技术要点

### 支持的 API 源
1. **lrclib.net** (主要)
   - URL: `https://lrclib.net/api/get`
   - 特点: 免费、稳定、国际化
   
2. **网易云音乐** (备用)
   - 搜索: `https://music.163.com/api/search/get/web`
   - 歌词: `https://music.163.com/api/song/lyric`
   - 特点: 中文歌曲覆盖率高
   
3. **QQ 音乐** (备用)
   - 搜索: `https://c.y.qq.com/soso/fcgi-bin/client_search_cp`
   - 歌词: `https://c.y.qq.com/lyric/fcgi-bin/fcg_query_lyric_new.fcg`
   - 特点: 流行歌曲丰富

### 降级策略
```
lrclib.net (优先)
    ↓ 失败
网易云音乐
    ↓ 失败
QQ 音乐
    ↓ 失败
返回错误
```

### 文件保存
- **位置**: 音乐文件同目录
- **格式**: `{歌曲名}.lrc`
- **备份**: `{歌曲名}.lrc.bak`

### 元数据获取优先级
1. ID3/FLAC 标签中的标题、艺术家
2. 文件名解析 (`艺术家 - 标题.mp3`)
3. 纯文件名作为标题

## ⚠️ 注意事项

### 前置条件
- ✅ 网络连接正常
- ✅ 音乐库已添加
- ✅ 歌曲有基本元数据

### 性能指标
- 单首歌曲(第1个源成功): ~1 秒
- 单首歌曲(需要多源): ~2-3 秒
- 100 首歌曲: ~3-5 分钟
- 1000 首歌曲: ~30-50 分钟

### 常见错误
- `所有歌词源均失败`: 三个 API 都未找到
- `未找到歌曲`: 搜索关键词不准确
- `请求失败`: 网络连接问题
- `API 返回错误状态码`: API 临时不可用

## 🎯 最佳实践

1. **先整理后下载**: 先使用"拆分歌词和音乐文件"功能
2. **检查元数据**: 确保歌曲有正确的标题和艺术家信息
3. **分批处理**: 大型音乐库可分多次下载
4. **保留备份**: 定期检查 `.lrc.bak` 文件
5. **选择时机**: 在网络较好时执行批量下载

## 🔄 故障排除

### 问题: 所有歌曲都下载失败
**解决**: 
1. 检查网络连接
2. 访问各 API 网站确认服务可用
3. 查看日志中的具体错误信息
4. 检查防火墙设置

### 问题: 部分歌曲找不到歌词
**解决**:
1. 检查歌曲元数据是否准确
2. 手动重命名文件为 "艺术家 - 标题.mp3" 格式
3. 在各个 API 网站搜索确认是否有该歌曲
4. 考虑手动添加歌词

### 问题: 下载速度太慢
**说明**: 
- 这是故意设计的速率限制保护
- 避免触发 API 封禁
- 多源尝试会增加时间但提高成功率
- 建议在空闲时间执行批量下载

### 问题: 网易云/QQ 音乐失败率高
**原因**:
- 这些 API 非官方公开接口
- 可能有反爬虫机制
- 建议依赖 lrclib.net 为主

## 💡 提示

- 下载过程中可以继续使用播放器其他功能
- 新下载的歌词会自动清除缓存,下次播放时立即生效
- 可以随时中断下载(关闭应用),已下载的歌词会保留
- lrclib.net 对国际歌曲支持最好
- 网易云和 QQ 音乐对中文歌曲覆盖率更高
- 多源策略显著提高了整体成功率

## 🎵 API 源选择建议

| 场景 | 推荐源 | 原因 |
|------|--------|------|
| 欧美歌曲 | lrclib.net | 国际化支持好 |
| 中文流行 | 网易云音乐 | 中文曲库丰富 |
| 华语经典 | QQ 音乐 | 老牌曲库完整 |
| 小众音乐 | 全部尝试 | 提高命中率 |
| 批量下载 | 默认多源 | 自动化处理 |
