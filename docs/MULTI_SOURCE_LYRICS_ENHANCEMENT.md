# 多源歌词下载功能增强

## 📋 更新概述

在原有 lrclib.net 单源歌词下载基础上,增加了**网易云音乐**和**QQ 音乐**两个备用 API 源,实现了**智能降级策略**,显著提高了歌词下载成功率。

## ✨ 新增功能

### 1. 多 API 源支持

#### lrclib.net (主要源)
- ✅ 免费开放 API
- ✅ 无需认证
- ✅ 国际化支持好
- ✅ 稳定性最高
- 🌍 适合: 欧美歌曲、国际音乐

#### 网易云音乐 (备用源 1)
- ✅ 中文歌曲覆盖率极高
- ✅ 歌词质量高
- ✅ 同步歌词完善
- ⚠️ 非官方 API,可能有反爬
- 🇨🇳 适合: 中文流行、华语音乐

#### QQ 音乐 (备用源 2)
- ✅ 流行歌曲资源丰富
- ✅ 华语经典曲库完整
- ✅ 支持 Base64 编码歌词
- ⚠️ 需要解码处理
- 🎵 适合: 华语流行、经典老歌

### 2. 智能降级策略

```
开始下载
    ↓
尝试 lrclib.net
    ├─ ✓ 成功 → 保存并返回
    └─ ❌ 失败 ↓
尝试 网易云音乐
    ├─ ✓ 成功 → 保存并返回
    └─ ❌ 失败 ↓
尝试 QQ 音乐
    ├─ ✓ 成功 → 保存并返回
    └─ ❌ 失败 ↓
返回错误(所有源均失败)
```

**优势**:
- 提高整体成功率(特别是中文歌曲)
- 自动化处理,用户无感知
- 保留备份机制,安全可靠

### 3. 技术实现

#### 核心函数

```go
// 多源降级下载(主入口)
func (lm *LyricManager) DownloadLyricWithFallback(
    trackPath, title, artist, album string

// 网易云音乐下载
func (lm *LyricManager) downloadFromNetease(
    title, artist string
) (string, error)

// QQ 音乐下载
func (lm *LyricManager) downloadFromQQMusic(
    title, artist string
) (string, error)

// Base64+Gzip 解码
func decodeBase64Gzip(encoded string) ([]byte, error)
```

#### API 端点

**网易云音乐**:
```bash
# 搜索歌曲
GET https://music.163.com/api/search/get/web?s={keyword}&type=1&limit=5

# 获取歌词
GET https://music.163.com/api/song/lyric?id={songID}&lv=1&kv=1&tv=-1
```

**QQ 音乐**:
```bash
# 搜索歌曲
GET https://c.y.qq.com/soso/fcgi-bin/client_search_cp?w={keyword}&format=json

# 获取歌词
GET https://c.y.qq.com/lyric/fcgi-bin/fcg_query_lyric_new.fcg?songmid={songMid}
```

## ⚠️ 注意事项

### API 稳定性

| API 源 | 稳定性 | 可用性 | 备注 |
|--------|-------|-------|------|
| lrclib.net | ⭐⭐⭐⭐⭐ | 99%+ | 最稳定,推荐依赖 |
| 网易云音乐 | ⭐⭐⭐ | 80-90% | 可能有反爬限制 |
| QQ 音乐 | ⭐⭐⭐ | 75-85% | 可能有反爬限制 |

### 法律合规
- lrclib.net: 明确提供开放 API
- 网易云/QQ 音乐: 非官方 API,仅供个人学习使用
- 建议: 优先使用 lrclib.net,备用源作为补充

### 网络要求
- 需要访问多个 API 域名
- 国内用户可能需要考虑网络环境
- 建议在网络良好时执行批量下载

## 🎯 最佳实践

### 1. 元数据质量
```
高质量元数据 → 高匹配率 → 高成功率
```
- 确保歌曲有正确的标题和艺术家
- 使用 ID3/FLAC 标签而非文件名
- 避免特殊字符和错误拼写

### 2. 分批处理
- 大型音乐库分多次下载
- 每次 100-200 首为宜
- 避免长时间占用网络

### 3. 时机选择
- 在网络状况良好时执行
- 避开网络高峰时段
- 可以在后台静默运行

### 4. 结果检查
- 查看通知中的统计信息
- 对于失败的歌曲手动处理
- 定期检查 `.lrc.bak` 备份文件

## 🔄 故障排除

### 问题: 网易云/QQ 音乐频繁失败

**可能原因**:
- API 接口变更
- 反爬虫机制触发
- 网络连接问题

**解决方案**:
1. 检查网络连接
2. 查看日志确认具体错误
3. 依赖 lrclib.net 为主
4. 等待一段时间后重试

### 问题: 下载速度过慢

**优化建议**:
1. 检查网络带宽
2. 减少同时运行的网络任务
3. 考虑分批次下载
4. 接受多源带来的额外时间

### 问题: 某些歌曲始终失败

**处理方案**:
1. 手动在各 API 网站搜索验证
2. 检查元数据准确性
3. 考虑手动下载歌词
4. 接受部分歌曲无歌词的现实
